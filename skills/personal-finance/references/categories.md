# Default Categories

Adapt to your local context. These are organized for Brazilian personal finance but
the structure works for any country.

## Expense Categories

### Moradia (Housing)
- Aluguel / Financiamento
- Condomínio
- IPTU
- Energia elétrica
- Água
- Gás
- Internet
- Manutenção residencial

### Transporte (Transport)
- Combustível
- Financiamento veículo
- Seguro veículo
- IPVA / Licenciamento
- Estacionamento / Pedágio
- Transporte público
- Manutenção veículo
- Aplicativo (Uber/99)

### Alimentação (Food)
- Supermercado
- Restaurante / Delivery
- Padaria / Café
- Lanches

### Saúde (Health)
- Plano de saúde
- Farmácia
- Consultas médicas
- Exames
- Odontologia

### Educação (Education)
- Escola / Faculdade
- Cursos / Livros
- Material escolar

### Lazer (Leisure)
- Entretenimento
- Viagens
- Streaming (Netflix, Spotify, etc.)
- Hobbies

### Pessoal (Personal)
- Vestuário
- Beleza / Estética
- Academia
- Presentes

### Financeiro (Financial)
- Tarifas bancárias
- Juros / Multas
- IOF
- Seguros (vida, residencial)
- Previdência privada

### Compras (Shopping)
- Eletrônicos
- Casa / Decoração
- Marketplace (MercadoLivre, Amazon, etc.)

### Família (Family)
- Mesada / Pensão
- Pet / Veterinário
- Babá / Escola filhos

### Trabalho (Work)
- Ferramentas / Software
- Coworking
- Equipamentos

### Igreja / Doações
- Dízimo
- Ofertório
- Doações

### Outros
- Não categorizado (catch-all — review periodically)

## Income Categories

### Receita (Income)
- Salário / Pró-labore
- Nota fiscal (PJ)
- Freelance
- PPR / Bônus
- Rendimento investimentos
- Dividendos
- Aluguel recebido
- Reembolso
- Cashback
- Crédito concedido (card credit)
